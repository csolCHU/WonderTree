# WonderTree
In fulfilment for completion requirements under the course IT0043 Web Development. 
WonderTree is a pet store company selling variety of items for dog, cat, fish, parrot, and chicken. Pet owners can now comfortably browse catalog in the comfort of their homes.

This website is a combination of using fundamental Web Design coding languages using HTML, CSS, and JavaScript. Bootstrap was used as a front-end framework for some website implementations. Images are sourced from stock images found online, through Google search engine. This website is non-profit and is for educational purposes only.
